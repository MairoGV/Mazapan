# generador qr en javascript puro

carpeta minimalista para crear códigos qr sin dependencias.

## archivos
- `index.html`  → interfaz
- `style.css`   → estilos
- `qr.js`       → lógica de generación

## uso local
1. clona o descarga la carpeta.
2. abre `index.html` en tu navegador.
3. escribe texto o url, pulsa "generar qr".
4. descarga el png resultante.

## demo en github pages
sube los archivos a un repositorio y activa github pages; visita `https://{usuario}.github.io/{repo}/`.
